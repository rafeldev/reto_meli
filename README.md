
# Reto Meli

## Built with 🛠️

- React
- React Router
- Axios
- SASS

## Installation 🔨

1. **Clone Project**: 
```
git clone https://github.com/rafeldev/reto_meli
```
3. **Move to project**: 
```
cd reto_meli
```

3. **Install dependencies**: 
```
npm install
```

3. **execute program**: 
```
npm run start
```


## API used
